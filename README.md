# weather-crawler-react-native
Weather crawler with React Native.

<br>

## How to Install
### 1. Git Clone
```
git clone https://github.com/Ajou-Team6/weather-crawler-react-native.git
cd weather-crawler-react-native
```

### 2. Dependency Install
```
npm install --save
```

### 3. Start Expo
```
npm start
```
